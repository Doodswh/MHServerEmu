Server Details

Server perks built for a faster, more active return.

The COA public server keeps the experience moving with tuned cooldowns, boosted progression, custom rewards, and quality-of-life adjustments that make logging back in feel worthwhile.

Features

Server Features

  • 50% Cooldowns: All hero skills have their cooldowns reduced by 50%, with more tuning planned for individual abilities over time.
  • Custom Commands: Access a range of custom and modified commands designed to expand gameplay options.

Economy

Economy

  • Eternity Splinter Conversion: Convert your Eternity Splinters into G's at a 20-to-1 ratio.

Rates

Rates

  • Experience Boost: Enjoy a 3x experience rate so leveling heroes feels meaningfully faster.
  • Loot Increase: Benefit from a 2x loot rate for better rewards across your runs.

Rewards

Daily Gifts

  • Custom Daily Gifts: A unique daily gift system, separate from the standard login rewards, adds a little extra reason to drop in regularly.
